# Portfolio Project
This Project is an SPA that has two parts: 
 - Portfolio page part
 - Admin part

##Technologies: 
- Vue.js
- Typescript
- SCSS,
- Vuex
- Firebase
- Vuetify.js

User have to type `/admin` in URI to  visit an Admin page
